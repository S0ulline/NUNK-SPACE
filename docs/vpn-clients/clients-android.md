# Клиенты для Android и Android TV

Операционная система Android предоставляет наибольшую свободу в настройке сетевой маршрутизации. Для неё существует огромное количество клиентов под любые нужды — от минималистичных решений для телевизоров до мощных инструментов для системных администраторов.

## 📱 Для смартфонов и планшетов (Android)

* **NekoBox (NB4A):** Настоящий швейцарский нож для продвинутых пользователей. Поддерживает переключение между криптографическими ядрами (Xray и Sing-box). Идеален для сложной маршрутизации и протоколов Hysteria 2 / VLESS.
* **Hiddify:** Кроссплатформенный клиент нового поколения на базе Sing-box. Максимально дружелюбный интерфейс с одной кнопкой. Отлично работает с подписками (Subscription URL) и сам подбирает самый быстрый узел.
* **v2rayNG:** Золотая классика. Строгий, аскетичный и невероятно стабильный клиент, работающий напрямую с ядром Xray. Идеален для гиков и тех, кто привык контролировать каждый параметр соединения.
* **v2Box:** Сбалансированный клиент с приятным современным интерфейсом. Отлично переваривает современные ссылки-ключи и поддерживает все актуальные методы маскировки трафика.
* **v2RayTun:** Простое и визуально чистое приложение. Хорошо подходит для базовых задач и пользователей, которым нужно быстро импортировать ключ и нажать кнопку "Подключиться".
* **Happ Proxy:** Легковесная и быстрая альтернатива с фокусом на моментальное подключение и минимальное потребление ресурсов аккумулятора.

## 📺 Для телевизоров и приставок (Android TV)

Настройка защиты трафика на смарт-ТВ (например, для просмотра зарубежных стриминговых сервисов) требует приложений, адаптированных под управление пультом.

* **Hiddify:** Благодаря интерфейсу с одной огромной кнопкой, это приложение идеально подходит для экранов телевизоров. Легко настраивается через сканирование QR-кода с телефона.
* **Happ Proxy:** Отлично оптимизирован для Android TV. Не перегружает слабые процессоры телевизионных приставок и стабильно держит фоновое соединение при просмотре тяжелого 4K-видео.

:::tip С чего начать?
Если вам нужно быстро настроить связь для себя или близких — ставьте **Hiddify**. Если вы хотите разделять трафик (например, пускать через сервер только YouTube, а остальное напрямую) — выбирайте **NekoBox**.
:::
