# Клиенты для Windows

Настройка защищенного соединения на полноценном ПК дает самые широкие возможности. Приложения для Windows могут работать в двух режимах: **System Proxy** (защита трафика браузеров) и **TUN Mode** (создание виртуальной сетевой карты, когда абсолютно весь трафик компьютера идет через ваш сервер).

## 🪟 Рекомендуемые клиенты

* **NekoRay:** Абсолютный лидер среди десктопных клиентов. Это мощная графическая оболочка на базе Qt. Позволяет в один клик переключаться между ядрами Xray и Sing-box. Имеет продвинутый режим TUN и позволяет создавать сложнейшие правила (например, пускать локальные ресурсы в обход сервера).
* **Hiddify (Desktop):** Если NekoRay — это панель управления самолетом, то Hiddify — это коробка-автомат. Клиент сам запрашивает права администратора, сам поднимает TUN-интерфейс и сам обновляет профили. Интерфейс на 100% идентичен мобильным версиям.
* **v2RayTun:** Отличный вариант с чистым и понятным интерфейсом для тех, кому функционал NekoRay кажется избыточным.
* **Happ Proxy:** Базовый и очень легкий клиент, не потребляющий лишнюю оперативную память компьютера.

:::warning О режиме TUN в Windows
Для работы режима глубокого туннелирования (TUN Mode) клиентам **всегда** требуются права Администратора, так как они создают в системе новую виртуальную сетевую карту. Всегда запускайте клиенты от имени Администратора, если планируете использовать этот режим.
:::
